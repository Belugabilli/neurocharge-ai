# 🔋 NeuroCharge AI

### Event-Driven Battery Intelligence System

---

## 📌 About

**NeuroCharge AI** is a real-time battery health monitoring system that uses **Machine Learning** to predict battery degradation and generate smart charging recommendations.

> Predict. Prevent. Prolong.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **Real-time Telemetry** | Live battery data (Temp, Voltage, Current, SOC) |
| 🧠 **ML Predictions** | Random Forest predicts State of Health (SOH) |
| ⚡ **Event Detection** | Thermal spikes, Fast charge, High SOC, Voltage anomalies |
| 📈 **Stress Score** | 0-100 scale indicating battery stress |
| 💡 **Smart Recommendations** | AI-generated charging tips |

---

## 🏗️ Tech Stack

**Frontend:** Next.js 14 + TypeScript + Tailwind CSS  
**Backend:** Flask + Python + scikit-learn  
**ML Model:** Random Forest Regressor (10,000+ samples)

## 🛠️ Local Setup

```bash
# Backend
cd backend
pip install -r requirements.txt
python app.py

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```

Open http://localhost:3000
